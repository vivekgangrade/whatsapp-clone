# WhatsApp Clone 📱

A high-fidelity WhatsApp Web clone built with UI excellence, featuring a modern, dark-themed responsive design. This project focuses on replicating the core user experience of WhatsApp Web with real-world interactions and aesthetics.

## ✨ Features

- **Responsive Chat Interface**: Pixel-perfect replication of WhatsApp Web's layout.
- **Dynamic Contact List**: Browse through mock contacts with a built-in search filter.
- **Interactive Messaging**: Send messages with real-time state updates.
- **Modern UI/UX**: 
  - Smooth transitions and hover effects.
  - Lucide React icons for a crisp look.
  - WhatsApp-consistent color palette (Teal and Dark Mode).
- **Message Bubbles**: Styled with time stamps and delivery status (mock).

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Library**: [React 19+](https://react.dev/)
- **Styling**: Vanilla CSS (High Performance)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app`: Contains the main application routes and UI components.
- `src/app/page.tsx`: The heart of the WhatsApp UI.
- `src/app/globals.css`: Custom design tokens and utility classes.
